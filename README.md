## 📖 О проекте

**FixCyrillic** — это простое и эффективное десктопное приложение на Python, созданное для решения специфической проблемы студентов и преподавателей **Байкальского государственного университета (БГУ)**.

При скачивании учебных материалов, методичек или документов с личного портала университета имена файлов часто отображаются некорректно («кракозябрами») из-за конфликта кодировок (`cp1252` вместо `cp1251`). Эта утилита мгновенно восстанавливает читаемые кириллические названия одним кликом.

> 💡 **Универсальность:** Хотя программа создана для нужд БГУ, она может быть полезна любому пользователю Windows, столкнувшемуся с проблемой перекодировки имен файлов из `Windows-1252` в `Windows-1251`.

## ✨ Возможности

- 🔄 **Мгновенное исправление:** Автоматическая конвертация имени файла из неверной кодировки в корректную кириллицу.
- ️ **Простой интерфейс:** Минималистичный GUI на Tkinter — ничего лишнего.
- 🛡️ **Безопасность:** Проверка на существование файла с таким именем перед переименованием (защита от перезаписи).
- 📂 **Выбор любого файла:** Работает с любым файлом в любой папке на компьютере.
-  **Обработка ошибок:** Корректное уведомление, если файл невозможно переименовать.

## 🚀 Как использовать

Программа максимально проста в использовании:

1.  Запустите исполняемый файл `.exe` или скрипт `main.py`.
2.  В открывшемся окне нажмите кнопку **«Выбрать и исправить файл»**.
3.  В диалоговом окне выберите файл с «неправильным» названием.
4.  Программа автоматически переименует файл в той же папке.
5.  Вы увидите сообщение об успешном переименовании.

### Меню «Справка»
В верхней панели доступно меню **«Справка»**, где можно найти краткую инструкцию и информацию об авторе.

## ⚙️ Технические детали

Для разработчиков и любопытных:

-   **Язык:** Python 3
-   **GUI:** Стандартная библиотека `tkinter` (не требует установки сторонних фреймворков).
-   **Логика перекодировки:**
    ```python
    # Преобразование байтов из cp1252 обратно в строку cp1251
    wrong_name.encode('cp1252').decode('cp1251')
    ```
-   **Зависимости:** Отсутствуют (используются только стандартные библиотеки `os`, `tkinter`).
